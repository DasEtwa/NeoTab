# NeoTab 1.2.0

`1.2.0` continues the ingame configuration work for NeoTab. It keeps the existing command and config behavior, restores the `1.1.1` color controls, and adds deeper GUI controls for tab colors, scoreboard presets, scoreboard performance, and the ActionBar Timer.

## Release Highlights

- Direct `Tab -> Colors` GUI with `purple`, `red`, `green`, `gold`, and custom hex input.
- Scoreboard line preset picker for online players, player name, ping, RAM, custom text, and clearing a line.
- Scoreboard presets can be saved, loaded, and deleted from the GUI or `/tab sb delete <name>`.
- Tab and scoreboard update intervals are configured separately.
- Scoreboard title animation can use the same styles as the tab header, but is configured independently.
- ActionBar Timer text is configurable with `{time}` and defaults to `timer ends` when finished.
- The active NeoTab color palette is reused by tab, scoreboard, timer, and NeoTab chat messages.

## Compatibility

- Platform: Paper
- Minecraft target: `1.20.6+`
- Build API: Paper `1.21-R0.1-SNAPSHOT`
- Java bytecode: Java 21
- PlaceholderAPI: optional
- LuckPerms: optional

## Control Panel

Open the GUI:

```text
/tab gui
```

Permission:

```text
neotab.gui
```

The GUI has three categories:

- `Tab`
- `Scoreboard`
- `Extras`

The GUI uses normal Bukkit inventories and cancels clicks/drags so players cannot take GUI items.

## Tab Menu

The `Tab` menu currently supports:

- `Name`: starts chat input for the tab/header name.
- `Style`: opens a selection menu for the existing animation styles.
- `Colors`: opens the four built-in color presets plus custom hex input.
- `Back`: returns to the main menu.

Chat input:

- Type the new value in chat.
- Type `cancel` to cancel.
- The value is applied live and saved to `config.yml`.

Existing commands still work:

```text
/tab setname <name>
/tab style <rainbow|purple-pulse|gradient-wave|static>
/tab color <purple|red|green|gold|#AA00AA,#BA55D3>
```

`/tab color` accepts the 1.1.1 presets and custom comma-separated hex colors. Permission: `neotab.color`.

## Scoreboard

The GUI supports:

- per-player scoreboard toggle
- line presets for online players, player name, ping, RAM, custom text, and clear
- saving the current scoreboard as a named preset
- loading saved presets
- deleting saved presets
- separate scoreboard update interval selection
- separate scoreboard title animation style

Config:

```yaml
scoreboard:
  enabled: false
  update-interval-ticks: 20
  title: "NeoTab"
  title-animation:
    enabled: true
    style: "static"
  lines:
    - "&7Online: &d{online}&7/&d{max}"
    - "&7Ping: &d{ping}ms"
    - "&7RAM: &d{ram_used}&7/&d{ram_max} MB"
  presets: {}
```

Commands:

```text
/tab sb on
/tab sb off
/tab sb toggle
/tab sb title <text>
/tab sb style <off|rainbow|purple-pulse|gradient-wave|static>
/tab sb interval <smooth|balanced|light|custom ticks>
/tab sb line <1-15> <text>
/tab sb clear <1-15>
/tab sb clearall
/tab sb save <name>
/tab sb load <name>
/tab sb delete <name>
/tab sb list
```

Permissions:

```text
neotab.scoreboard
neotab.scoreboard.toggle
neotab.scoreboard.edit
neotab.scoreboard.presets
```

Supported built-in placeholders:

```text
{online}
{max}
{ping}
{avg_ping}
{ram_used}
{ram_max}
{ram_percent}
{server_name}
{player}
{player_name}
```

PlaceholderAPI placeholders in scoreboard title and lines are supported when PlaceholderAPI is installed and `enable-placeholderapi` is true.

## Extras

The `Extras` menu currently supports:

- `Tab Interval`: uses the existing performance presets (`smooth`, `balanced`, `light`).
- `Scoreboard Interval`: uses the existing performance presets independently from the tab interval.
- `ActionBar Timer`: opens simple timer controls, including fixed starts and custom duration chat input.

No new performance preset system was added. The GUI reuses the existing preset names, but tab and scoreboard store their selected intervals independently.

## ActionBar Timer

Config:

```yaml
extras:
  actionbar-timer:
    enabled: true
    running-format: "{time}"
    paused-format: "Paused {time}"
    ended-format: "timer ends"
```

Commands:

```text
/tab timer start <duration>
/tab timer stop
/tab timer pause
/tab timer resume
/tab timer text <text with {time}>
```

Duration examples:

```text
30s
5m
10m
1h
```

Permissions:

```text
neotab.timer
neotab.timer.admin
```

The timer is per-player, updates once per second, uses the ActionBar, uses the active NeoTab color palette, and is cleaned up when the player leaves.

The GUI can start fixed 5 minute and 10 minute timers or ask for a custom duration in chat.

## Intentionally Minimal

Not included in this `1.2.0` release:

- BossBar timer
- Drag-and-drop GUI editing
- Web dashboard
- Database storage
- Legacy support below `1.20.6`
- Large adapter rewrite

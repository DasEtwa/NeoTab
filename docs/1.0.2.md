# NeoTab 1.0.2

`1.0.2` is the original stable public release of NeoTab.

## Compatibility

- Platform: Paper
- Minecraft versions used on Modrinth: `1.21.11`, `26.1.x`
- Java: use the Java version required by your Paper server

## Features

- Animated tablist header
- Configurable server name
- Live RAM footer
- Personal player ping
- Average server ping
- Optional LuckPerms prefix/suffix support
- `/tab reload`
- `/tab setname <name>`
- `/tab style <rainbow|purple-pulse|gradient-wave|static>`

## Configuration

NeoTab creates its config at:

```text
plugins/NeoTab/config.yml
```

Common options:

```yaml
server-name: "<gradient:#AA00AA:#BA55D3>Mein Epic Server</gradient>"
animation-style: "rainbow"
update-interval-ticks: 3
enable-luckperms-prefix: true
```

Footer placeholders provided by NeoTab:

```text
{used}
{total}
{percent}
{online}
{max}
{playerPing}
{avgPing}
```

## Notes

- This release does not include PlaceholderAPI support.
- The source for this exact release was reconstructed later from the original JAR.
- For PlaceholderAPI placeholders, use `1.1.0-Beta.1` or newer.
